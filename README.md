# ---Trabalhos Escolares---

Este repositório contém algoritmos e pequenos projetos desenvolvidos
durante meus estudos de programação.

O objetivo é praticar lógica de programação, estruturas básicas,
resolução de problemas, organização de código e enriquecer meu portifólio.

Os exercícios são implementados principalmente em <strong>Python</strong> e podem
incluir desafios simples, cálculos matemáticos e pequenos programas
interativos.

## Algoritmos presentes:

<ul>
  <li>Conversão de Graus</li>
  <li>Cálculo do IMC</li>
  <li>Busca em largura</li>
  <li>Fatorial</li>
</ul>
